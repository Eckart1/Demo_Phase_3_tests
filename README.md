Setup:
Ensure that when downloading file that you move it to your PycharmProjects folder where all your Pycharmprojects would be stored.
If you are not sure where to find your PyCharm folder, it is usually under User/YourPCName/PycharmProjects

Run Commands:
When the project is installed and moved to the correct file you can just open it and run the project by typing py.test in the terminal. 


Assumptions:
It is assumed that you have PyCharm installed
It is assumed that you have selenium, py.test and pip3 installed
